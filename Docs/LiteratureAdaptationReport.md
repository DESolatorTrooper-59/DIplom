# Отчет об адаптации списка литературы под контекст проекта

Дата подготовки: 01.06.2026

## Контекст отбора

Список литературы адаптирован под фактический технологический и предметный контекст проекта:

- настольное приложение для управления спортивными турнирами;
- пользовательский интерфейс на WPF;
- платформа .NET Framework 4.8 и язык C#;
- хранение и обработка данных через SQL Server, `System.Data`, ADO.NET и `SqlConnection`;
- CRUD-операции, импорт/экспорт данных, каталог турниров, участников и команд;
- построение и редактирование турнирной сетки;
- роли пользователей: гость, игрок, организатор, администратор;
- аутентификация, хеширование паролей и разграничение доступа.

Из списка удалены источники, которые относятся к другой технологической платформе, другому типу приложения, неподходящей СУБД, слабым справочным сайтам либо дублируют уже оставленные учебные источники.

## Принцип сортировки

Итоговый список отсортирован не по алфавиту, а по логике раскрытия проекта:

1. Сначала идут источники по жизненному циклу, методологиям, ООП и информационным технологиям.
2. Затем источники по основному языку и платформе: C#, .NET Framework, Visual Studio.
3. После этого выделен блок пользовательского интерфейса WPF: WPF, XAML, привязка данных, элементы управления.
4. Далее расположен блок работы с данными: SQL, базы данных, ADO.NET, `SqlConnection`, SQL Server, SSMS, резервное копирование.
5. В конце собраны источники по надежности, безопасности, ролям, разрешениям и аутентификации.

Такая сортировка удобна для дипломного проекта: список литературы повторяет путь от общих основ разработки к конкретной реализации приложения и затем к вопросам надежности и безопасности.
